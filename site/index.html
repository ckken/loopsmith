Loopsmith

面向 Codex 的自动化修复编排工具

用可审计的 review、repair、verify 循环， 把 AI 生成的候选修复推进到可验证结果。

Lifecycle Hooks

在关键生命周期点注入自定义命令，保障质量与合规。

[hooks]
before_review = "cargo fmt --all"
after_repair  = "cargo clippy --all"
after_verify  = "cargo test --locked"

多角色 Agents

使用不同 Codex 模型与子智能体，按角色分工高效协作。

[agents]
review.model = "gpt-5.4"
repair.model = "gpt-5.4-mini"
verify.model = "gpt-5.3-codex-spark"

可审计的执行记录

完整记录每一轮输入输出、变更与验证结果，便于追溯与复盘。

runs/<timestamp>/
├─ round-1-review/record.json
├─ round-2-repair/record.json
├─ round-3-verify/record.json
└─ summary.json

安全与可控

支持沙箱隔离、审批策略与只读模式，掌控风险边界。

[safety]
sandbox         = "workspace-write"
approval_policy = "on-failure"
readonly_paths  = ["src/**", "tests/**"]
